# 작업 계획서 — 워치리스트 기능

## 1. 개요

관심 종목을 매수 전 단계에서 등록·모니터링하고, 매수 결정 시 포트폴리오로 자동 편입하는 기능. 본 계획서는 정식 개발을 위한 구체화 작업 범위를 정의한다.

### 목적

- 희망 매수 구간 모니터링으로 진입 타이밍 포착

---

## 3. 기능 상세

### 3-1. 워치리스트 등록

**입력 항목**

| 필드               | 필수 | 설명                            |
| ------------------ | ---- | ------------------------------- |
| 종목명             | ✅   | 직접 입력                       |
| 티커               | ✅   | KRX: 6자리 숫자 / US: 영문 심볼 |
| 시장               | ✅   | KRX / US 선택                   |
| 편입 예정 섹터     | ✅   | 기존 섹터 중 선택               |
| 편입 예정 비중 (%) | ✅   | 해당 섹터 내 목표 비중          |
| 매수 희망 하단가   | 선택 | 진입 구간 하한선                |
| 매수 희망 상단가   | 선택 | 진입 구간 상한선                |

### 3-2. 진입 구간 상태 표시

현재가와 희망 매수 구간을 실시간 비교해 3가지 상태로 표시한다.

| 상태     | 조건                       | 배지                  |
| -------- | -------------------------- | --------------------- |
| 진입가능 | `하단가 ≤ 현재가 ≤ 상단가` | ✅ 진입가능 (초록)    |
| 대기중   | `현재가 > 상단가`          | ▲ 구간까지 +X% (회색) |
| 하단이탈 | `현재가 < 하단가`          | ▼ 하단이탈 -X% (노랑) |
| 미설정   | 구간 미입력                | — (표시 없음)         |

**진입가능 상태 강조**

- 행 배경색 변경 (초록 틴트)
- 탭 배지 숫자와 별도로 "진입가능 N개" 카운트 표시 예정
